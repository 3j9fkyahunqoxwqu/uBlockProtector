Installation instructions

1. Check if your setup is supported, it is highly recommended that you configure your setup so it is officially supported by us. You may still try to continue even if your setup is not officially supported, but things might not work properly.

2. Enable Adblock Warning Removal List from your AdBlocker settings panel.

3. Install AdBlock Protector from install links below.


Setups we support:

Browser: Chrome (desktop)

Note: FireFox is known to not work.

Script host: Tampermonkey (desktop)

Note: Greasemonkey is known to not work.

AdBocker: uBlock Origin (desktop)

Note: AdBlock, Adblock Plus, and Adguard are known to not work.


Android support coming soon!


Adblock Protector install links:

1. Click here to install Script.

2. Activate Adblock Protector List from your AdBlocker settings page. Can't find it? Click here to install it manually.

Note: You need both Script and List.

Why is it called "AdBlock Protector"?

When the project started, it was aimed at Adblock Plus. But we decided to switch to uBlock Origin after testing out its amazing features.

We are seriously considering renaming this project to "uBlock Protector".
