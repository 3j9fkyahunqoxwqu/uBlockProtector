Installation instructions

You must be using Chrome and uBlock Origin to use this project. Chromium and uBlock Origin should work as well.

Do not want to switch your setup? Check out AAK-Cont!


Step 1. Enable Adblock Warning Removal List from uBlock Origin settings panel.

Step 2. Enable uBlock Protector List from uBlock Origin settings page. Can't find it? Click here to install it manually.

Step 3. Install uBlock Protector Extension

Note: This extension covers uBO-Extra's functionalities.
