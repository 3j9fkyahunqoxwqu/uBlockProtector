Why use AdBlockers?

Ads are important for websites to sustain themselves. However, some advertiser use ads to track users' activities and sometimes use exploits to install malwares onto users' devices, and an AdBlocker can stop these from happening.

Recently, many websites start to employ methods to detect and sometimes track AdBlockers and ask or sometimes even force the user to disable them. The good news is, there are ways to protect AdBlockers from being detected.

How to protect my AdBlockers?

1. Adblock Warning Removal List (Check your AdBlocker settings)

2. Anti-Adblock Killer

3. This project: AdBlock Protector

Make sure to install AdBlock Protector LAST for compatibility!

When a protection mechanism is implemented into one of the larger projects, it will be removed from this one. This project is meant to be a quick solution while waiting for larger projects to update. You should keep all of these projects installed.

Why not create pull requests to existing projects?

It will take time before it gets added, and our goal is quick solutions. We will send our solutions to larger projects such as Anti-Adblock Killer.

Some of our codes can be unstable or inneficient, but that is the tradeoff for quick solutions.

How do I use it?

1. Install a script host and an AdBlocker, if you do not have them installed already. I will make sure that this project works on Chrome and Opera with Tampermonkey and Adblock Plus, it should work on most setups, but for now, I will not offer support for other setups in case it does not work.

2. Install our Userscript.

3. Install our List.

Note: You need both the Userscript and the List.

4. See if there is something you need to know about your website.


If you are using Greasemonkey on FireFox, use this dropper instead of the Userscript above.

How do I report a bug or a website?

Please open an issue if you have a website, a bug, a typo, or a suggestion to report.

If you are reporting a website, please provide the link to the website in question, and a screenshot, also, you should report it to larger projects as well.
