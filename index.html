Why use AdBlockers?

Ads are important for websites to sustain themselves. However, some advertiser tracks your activities and sometimes use exploits to install malware onto your devices, and an AdBlocker can stop these from happening.

Recently, many websites start to use methods to detect and sometimes track AdBlockers and ask or sometimes even force you to disable them. The good news is, there are ways to protect AdBlockers from being detected.

How to protect my AdBlockers?

1. Enable Adblock Warning Removal List (Check your AdBlocker settings)

2. Install AdBlock Protector

How do I use it?

1. Check if your setup is supported, it is highly recommended that you configure your setup so it is officially supported by us. You may still try to install this project if your setup is not supported, but it might not work properly.

2. Install from install links below.
