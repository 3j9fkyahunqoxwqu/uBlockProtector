Why use AdBlockers?

Ads are important for websites to sustain themselves. However, some advertiser tracks your activities and sometimes use exploits to install malware onto your devices, and an AdBlocker can stop these from happening.

Recently, many websites start to use methods to detect and sometimes track AdBlockers and ask or sometimes even force you to disable them. The good news is, there are ways to protect AdBlockers from being detected.

How to protect my AdBlockers?

1. Adblock Warning Removal List (Check your AdBlocker settings)

2. Anti-Adblock Killer

3. This project: AdBlock Protector

Make sure to install AdBlock Protector LAST for compatibility!

When a protection mechanism is implemented into Anti-Adblock Killer, it will be removed from this one. This project is meant to be a quick solution while waiting for Anti-Adblock Killer to update. You should keep all of these projects installed.


Looks like Reek, the author of Anti-Adblock Killer disappeared again, should we merge Anti-Adblock Killer into this project? Let me know.

Why not create pull requests to existing projects?

It will take time before it gets added, and our goal is quick solutions.

Some of our codes can be unstable or inefficient, but that is the tradeoff for quick solutions.

How do I use it?

1. Check if your setup is supported, it is highly recommended that you configure your setup so it is officially supported by us. You may still try to install this project if your setup is not supported, but it might not work properly.

2. Install from install links below.


Setups we support:

Browser: Opera (desktop), Chrome (desktop), Firefox (desktop)

Script host: Tampermonkey (desktop), Greasemonkey(unsafe) (desktop)

AdBocker: Adblock Plus (desktop)
